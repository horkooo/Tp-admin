网站名称，显示在浏览器标签上。调用方法：{literal}{:sysconf('site_name')}{/literal}

程序的版权信息设置，在后台登录页面显示。调用方法：{literal}{:sysconf('site_copy')}{/literal}

当前程序名称，在后台主标题上显示。调用方法：{literal}{:sysconf('app_name')}{/literal}

当前程序版本号，在后台主标题上标显示。调用方法：{literal}{:sysconf('app_version')}{/literal}

百度统计应用ID，可以在百度网站统计申请并获取。调用方法：{literal}{:sysconf('tongji_baidu_key')}{/literal}

网站备案，显示网站底部。调用方法：{literal}{:sysconf('site_beian')}{/literal}

首页标题，网站SEO设置。调用方法：{literal}{:sysconf('site_stitle')}{/literal}

首页关键词，网站SEO设置。调用方法：{literal}{:sysconf('site_skeywords')}{/literal}

首页描述，网站SEO设置。调用方法：{literal}{:sysconf('site_sdesc')}{/literal}

上传图片

建议上传ICO图标的尺寸为128x128px，此图标用于网站标题前，ICON在线制作。调用方法：{literal}{:sysconf('browser_icon')}{/literal}
